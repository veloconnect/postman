# Veloconnect – Request Collection

Bruno collection with example requests for the Veloconnect B2B API standard.

## What is Veloconnect?

Veloconnect is an open standard for data exchange between bicycle retailers 
and suppliers. It is maintained by DiBike – Digitize Bike Business Group.

More information: [veloconnect.de](https://www.veloconnect.de)

## Modules

| Module | Request |
|---|---|
| Profile | GetProfileRequest |
| Search | TextSearchRequest, SearchResultRequest |
| Item | GetItemDetailsRequest |
| Classification | GetClassification |

## Getting started

1. Install [Bruno](https://www.usebruno.com)
2. Clone or download this repository
3. Open the collection in Bruno: *Open Collection* → select folder
4. Copy `environments/template.yml` and add your credentials
5. Run requests

## Requirements

- Bruno (free, open source)
- Veloconnect server access (ServerURL, BuyersID, Password)
